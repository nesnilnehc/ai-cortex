# 主动建议表（阶段→技能）

本表用于在不同协作阶段给出建议技能入口。  
说明：该表为建议映射，不替代任务语义匹配。

| 阶段 | 建议技能 | 说明 |
| :--- | :--- | :--- |
| 需求澄清 | `analyze-requirements` | 将模糊意图转为可验证需求 |
| 方案设计 | `design-solution` | 形成架构与数据流设计文档 |
| 任务拆解 | `breakdown-tasks` | 生成可执行任务与依赖 |
| 文档治理 | `audit-docs` | 全量审计规范、结构与 SSOT |
| 结构整理 | `tidy-repo` | 检测并清理仓库结构问题 |
| 文档评估 | `assess-docs` | 做合规与就绪度评估 |
| 代码审查 | `review-code` | 统一编排多维度代码审查 |
| 提交交付 | `commit-work` | 生成规范提交并附质量检查 |
