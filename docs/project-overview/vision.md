# Vision

A world where agent capabilities are as easy to discover, reuse, and govern as shared code — so teams scale reliable agent-driven work without fragmentation or ad hoc prompt logic.

- **Aligned with mission:** This vision extends the mission (discoverable, reusable, governable; scale reliable agent-driven work) into a long-term future state.
- **Time horizon:** Long-term; the capability layer becomes shared infrastructure across teams and ecosystems.
