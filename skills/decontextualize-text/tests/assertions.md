# Skill Verification Assertions: decontextualize-text

请对产出的内容按以下准则进行严格审计（判分制：0 或 1）：

| 准则标号 | 描述 | 检测逻辑 |
| :--- | :--- | :--- |
| **A-01** | **去特异化** | 是否仍包含具体的公司名、项目代号（如 Project-X）、内部协议名或人名？ |
| **A-02** | **隐私保护** | 是否包含手机号、UID、详细地址或任何可定位到特定真实个体的特征？ |
| **A-03** | **逻辑保真** | 剔除背景后，其核心技术方法论、故障现象或业务逻辑是否依然清晰连贯？ |
| **A-04** | **无偏见表述** | 语气是否调整为客观、通用的技术描述，而非内部口吻？ |

### 评估公式

- **Pass**: A-01 == 1 && A-02 == 1 && A-03 == 1
- **Score**: (A-01 + A-02 + A-03 + A-04) / 4
