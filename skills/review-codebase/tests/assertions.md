# Skill Verification Assertions: review-codebase

| 准则标号 | 描述 | 检测逻辑 |
| :--- | :--- | :--- |
| **B-01** | **范围一致** | 评审范围是否与用户指定的路径/目录一致，未擅自限于 diff？ |
| **B-02** | **维度覆盖** | 是否围绕「现状/架构/设计」覆盖架构与边界、设计模式与一致性、技术债与可维护性、跨模块依赖与耦合、安全与性能现状中的关键点？ |
| **B-03** | **可定位** | 指出问题时是否带有 file:行 等可定位引用？ |
| **B-04** | **可落地** | 是否对重要问题给出了具体修复或重构建议？ |

### 评估公式

- **Pass**: B-01 == 1 && B-02 == 1 && B-03 == 1 && B-04 == 1
- **Score**: Σ(B-n) / 4
