# Skill Verification Assertions: write-agents-entry

请对产出的 AGENTS.md 或修订建议按以下准则进行判分（0 或 1）；依据为本技能 [SKILL.md](../SKILL.md) 内嵌的「产出契约：AGENTS.md 撰写规范」章节。

| 准则标号 | 描述 | 检测逻辑 |
| :--- | :--- | :--- |
| **W-01** | **会节完整性** | 产出是否包含：开篇、项目身份、权威来源、行为约定、发现与加载（概要）、语言与沟通、引用表？若为修订建议，是否针对缺失会节给出补全说明？ |
| **W-02** | **三要素** | 是否明确体现项目身份（一句话 + 资产类型/目录/定义规范）、权威来源（定义/目录/使用契约在何处）、行为约定（至少 2 条可执行约定）？ |
| **W-03** | **引用表** | 引用表是否至少包含：规范来源、本手册 Raw URL（若适用）、定义与测试/使用与安装 spec、入口索引？路径是否可解析？ |
| **W-04** | **可执行约定** | 行为约定是否使用「应当」「须」「不得」等明确措辞，且每条可对应到 spec 或文档？ |
| **W-05** | **无冗余** | 是否避免在 AGENTS.md 内重复 spec/docs 的完整正文，仅做索引与摘要或指路？ |
| **W-06** | **输入匹配** | 产出的项目身份、资产表、路径是否与输入（项目定位、资产类型与目录）一致，无臆造或不符？ |

### 评估公式

- **Pass**: W-01 == 1 && W-02 == 1 && W-03 == 1 && W-06 == 1
- **Score**: (W-01 + W-02 + W-03 + W-04 + W-05 + W-06) / 6
