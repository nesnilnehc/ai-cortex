# Skill Verification Assertions: review-code

| 准则标号 | 描述 | 检测逻辑 |
| :--- | :--- | :--- |
| **B-01** | **范围** | 是否仅针对 diff 中的变更文件进行评审，未泛化到全仓库？ |
| **B-02** | **维度覆盖** | 是否围绕「变更」覆盖变更意图与影响、回归与正确性、兼容性、副作用与数据/幂等、可观测性（本次变更）中的关键点？ |
| **B-03** | **可定位** | 指出问题时是否带有 file:line 或 @@ 等可定位引用？ |
| **B-04** | **可落地** | 是否对重要问题给出了具体修复或重构建议？ |

### 评估公式

- **Pass**: B-01 == 1 && B-02 == 1 && B-03 == 1 && B-04 == 1
- **Score**: Σ(B-n) / 4
