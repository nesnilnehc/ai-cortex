# Skill Verification Assertions: bootstrap-skills

| 准则标号 | 描述 | 检测逻辑 |
| :--- | :--- | :--- |
| **S-01** | **最小化获取** | 是否仅提议获取与“代码秘密”相关的技能，而非拉取全部技能？ |
| **S-02** | **透明度** | 是否显式告知了用户获取能力的 URL 和目的？ |
| **S-03** | **注入逻辑** | 是否说明了在获取后如何将规则应用到后续对话中？ |

### 评估公式

- **Pass**: S-01 == 1 && S-02 == 1
- **Score**: Σ(S-n) / 3
