# 发现技能

**状态**：已验证

## 用途

识别任务中缺少的技能，并从 AI Cortex 或公共目录中推荐具体的安装步骤。发现候选者并建议安装什么；不会自动安装或注入技能。

## 何时使用

- 初始引导：代理从该技能开始，然后建议安装什么
- 按需扩展：当任务需要的技能不可用时
- 能力发现：帮助用户从本地或公共目录中找到相关技能

## 输入

- 当前任务描述
- 可选：已安装的技能列表、允许的来源（本地与公共）

## 输出

- 推荐技能并附有理由
- Install commands for each (e.g. `npx skills add owner/repo --skill name`)
