# 生成代理条目

**状态**：已验证

## 用途

根据嵌入式输出合约在存储库根写入或修改 AGENTS.md。建立项目身份、权威来源和行为期望。可执行的期望（必须/应当/不得）；规格和索引链接的参考表。

## 何时使用

- 新项目：为没有 AGENTS.md 的存储库添加代理条目
- 修改现有内容：审核并完成缺失的部分
- 采用格式：将输出合同用于其他项目
- 合规检查：对照合同审核，输出修改建议

## 输入

- 一线定位
- 顶级资产和目录
- 可选：AGENTS.md 原始 URL、现有条目、主要语言

## 输出

- 完整的 AGENTS.md（或差异/修订文本）
- 或审核清单和修改建议
