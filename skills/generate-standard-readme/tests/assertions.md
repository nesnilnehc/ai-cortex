# Skill Verification Assertions: generate-standard-readme

| 准则标号 | 描述 | 检测逻辑 |
| :--- | :--- | :--- |
| **B-01** | **结构完整性** | 是否包含：摘要、安装、快速开始、贡献指南、协议五个核心板块？ |
| **B-02** | **输入匹配性** | 是否正确提取了项目中声明的运行环境（如 Go 1.21）和核心功能？ |
| **B-03** | **文档规范性** | 是否使用了正确的 Markdown 语法（如代码块、徽章链接、层次有序）？ |
| **B-04** | **工程调性** | 语气是否专业，是否避免了使用如 'My first project' 这种非专业用语？ |

### 评估公式

- **Pass**: B-01 == 1 && B-02 == 1 && B-03 == 1
- **Score**: Σ(B-n) / 4
